## 🖼️ Adding Your Profile Photo

Place your professional headshot named **`profile.jpg`** in the root folder.  
Recommended: square format, minimum 400×400px, professional background.

---

## 🛠️ Local Development

```bash
# Clone the repo
git clone https://github.com/tsitohainaraz/portfolio.git
cd portfolio

# Install dependencies
pip install -r requirements.txt

# Run locally
streamlit run app.py
```

---

## ☁️ Deploy on Streamlit Cloud (Free)

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **"New app"**
4. Select your repository → `app.py`
5. Click **"Deploy"** ✅

Your portfolio will be live at:  
`https://your-username-portfolio.streamlit.app`

---

## 🎨 Customization

Edit `app.py` to update:
- Personal info, experience, projects
- Skill percentages
- Color accents (CSS variables at top of `inject_css()`)
